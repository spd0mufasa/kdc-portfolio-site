# AI System Project

This project is a backend-first, architecture-heavy system designed to ingest, process, and retrieve medical documents securely for multiple users. It includes:

- A multi-user ingestion pipeline  
- A vector-database-powered semantic retrieval engine  
- A secure API layer for agents  
- A future GUI for interacting with the system
